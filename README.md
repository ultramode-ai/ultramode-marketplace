# UltraMode plugin marketplace

Official marketplace for the UltraMode Claude Code plugin.

## Install

```
/plugin marketplace add cloudwishcom/ultramode-marketplace
/plugin install ultramode@ultramode
```

Or in VSCode "Manage Plugins" → Marketplaces tab, paste:

```
cloudwishcom/ultramode-marketplace
```

## What's in the plugin

- **106 MCPs** via mcp-router (GitHub, Stripe, Hostbill, n8n, Playwright, Kubernetes, and 100+ more)
- **Remote Workspace** — VS Code-grade SSH for AI agents (auto-deployed daemon, ~6ms hot reads)
- **Vault** — AES-256-GCM encrypted memory + connections (local unlimited / free 100 MB / pro 10 GB)
- **42 skills** — ads (Google/Meta/TikTok/LinkedIn), PDF, funnels (CRO/copy/email), frontend-design, debugging
- **30 slash commands** — `/ultramode-remote-connect`, `/ultramode-oauth-login`, `/ultramode-help`, etc.

Plugin package: [`@ultramode/install` on npm](https://www.npmjs.com/package/@ultramode/install).
